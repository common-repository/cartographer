== Description ==

Easily add a custom Google map to your WordPress site. Multiple locations on a single map with ease.

= Features =

* Add locations, like offices, stores, etc
* Easy shortcode builder helps you build and embed your custom map on any page or post
* Location Import/Export now available as an addon!

= Coming Soon =
* Filterable Map Pins
* Store Locator

= Please Vote and Enjoy =

Your votes really make a difference! And don't forget to leave your support/feature requests in the forums!

== Installation ==

1. Upload 'cartographer' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add new locations under the Cartographer nav item in your WordPress admin
4. Use the shortcode builder on your page or post to embed your map, it's that easy!
